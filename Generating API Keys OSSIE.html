Generating API Keys OSSIE

Source: https://support.officespacesoftware.com/s/article/Managing-API-Keys-HC

 

To Generate an API Key

 

  1. Open the Hamburger Menu, shown as three lines in the top right corner of the OfficeSpace screen

 

  1. Open the Admin Tab in the Hamburger menu

 

  1. Select OfficeSpace API to be taken to the Manage API Keys page

 

  1. Select the API Keys section

 

  1. Select Generate API Key to generate a new key by following the directions in the New API Key pop-up
